CSS Units: em, rem, px, and % Examples

1. px

Pixels are absolute units representing a fixed size on the screen.

Width: 200px, Height: 50px, Font-size: 16px

2. em

The em unit is relative to the font size of the element's parent.

Width: 12em, Height: 3em, Font-size: 20px

3. rem

The rem unit is relative to the root element's font size (usually the html element).

Width: 12rem, Height: 3rem, Font-size: 1.5rem (24px)

4. %

Percentage units are relative to the size of the parent element. For width and height it is relative to the parent's width and height respectively. For font-size it is relative to the parent's font size.

Parent container width: 400px
Width: 50%, Height: 50px, Font-size: 100%
